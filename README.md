# 🌌 Nebula Forge (星云锻造炉)

**Nebula Forge** 是一款基于 **Tauri + React** 构建的本地 AI 网页构建器。它完全基于 **Ollama** 驱动本地大模型（如 Qwen2.5-Coder），为你提供隐私、安全、零成本的流式代码生成与实时渲染体验。

### ✨ 核心特性

- 🧠 **纯本地智能内核**：深度集成 Ollama，数据完全本地化运行，无需联网，隐私绝对安全。
- ⚡ **流式代码跳字**：搭载优化的流式解析器，AI 输出的同时，代码在右侧**实时同步渲染**。
- 🌈 **智能代码高亮**：支持动态语言嗅探，无论生成 HTML 还是 Python，都能精准识别并实时高亮。
- 👁️ **即时沙盒预览**：内置安全沙盒环境，生成的代码一键切换为可视化交互网页。
- 📋 **一键纯净提取**：带有成功状态反馈的“一键复制”功能，生成的代码随用随取。
- 🎛️ **极简专注布局**：沉浸式工作台，底部固定输入区与自适应折叠侧边栏。

### 📦 下载与使用 (小白必看)

如果你只想使用本软件，无需配置复杂的代码环境，请按以下步骤操作：

1. **下载软件**：前往右侧的 **[Releases 页面](https://github.com/luyao001/nebula-forge/releases)** 下载最新的 `.exe` 安装包并安装。
2. **安装 AI 大脑**：确保你的电脑已安装 [Ollama](https://ollama.com/)，并在系统终端运行 `ollama pull qwen2.5-coder:7b` 拉取模型。
3. **开启跨域权限 (非常重要)**：打开 Windows 终端 (cmd)，执行命令 `setx OLLAMA_ORIGINS "*"`。执行完毕后，请**彻底退出并重启 Ollama**（在右下角系统托盘右键退出，再重新打开）。
4. 打开 Nebula Forge，开始你的本地网页锻造之旅！
